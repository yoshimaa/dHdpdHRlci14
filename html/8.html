あとがき

あとがき

このプロジェクトは大変、短期間のプロジェクトでした。

旧ツールはneruのただの思い付きで作成したアイデア商品でしたが、新しいツールはneruの個人開発となりました。

旧ツールは依然として問題なく動作します。ただし、Twitter APIの有料化より、少々月額費用がかかるようになってしまいました。そのため、月額費用が不要な新しいツールを開発しました。また、4年ほど販売して旧ツールに、いろいろな要望が寄せられましたが、旧ツールはシステム上実現できないものが多くありました。しかし、今回はGAS（Google Apps Script）を用いて1からコードを書いているためより柔軟な対応が可能となりました。

また、X(Twitter) API Freeはひとつのアプリしか利用できない制約があり、X(Twitter) APIを使ったWordPressとのプラグインの併用に問題があったため、オールインワンで使えるツールを開発しました。

いろいろなことが積み重なりがあり、今回のツールが生まれたわけです。イーロンマスクさんのTwitter APIの有料化がなければ、このツールは生まれなかったでしょう。

Twitter社がX社と名称を改めたことを受け、当ツール名もCraftTweet（クラフトツイート）からCraftTwix（クラフトツイックス）に変更!

「Craft」という言葉がよさそうでした。Craftという職人が技をこらして作りだすというニュアンスが好きです。Craftというと手作り感があり、ものづくりとは部品を組み合わせて作るところがあるため、当ツールの組み合わせ機能などを表現するのによい名前という気がしましたね。

X社と変わりましたが、X(Twitter)を眺めると青い鳥が好きな人がたくさんいましてね。僕自身も青い鳥に愛着があります。それなら、昔からTwitterが好きな人のためにTwiは残すべきではないかと。ちゃんと「x」はつけていますから、X社に対する尊重と今風にすることを忘れていません。Xという文字が象徴するように、未知数のソフトになるといいですね。少なくともTwitter Api Freeの魅力を最大限引き出そうと試みました。

TwitterからXの架け橋となり、偏りがない調和があり平和な名前にしたつもりです。TwitterからXへの変遷を示しており、その歴史を「twix」という名前で表現したつもりです。

マニュアルの電子書籍

余談ですが、少数派かもしれませんが、このマニュアルが興味を持たれた方もいるかもしれません。こちらも小ネタでやっているneruの個人開発の1つとして進めています。電子書籍プロジェクトです。

簡単にいうと、note（ノート）のような作品を配信するプラットフォームが存在しますが、BOOTH（ブース）やStripeなどを通して、どこでもローカル上で配布できるものを開発しています。実現したかったことは次のようなことです。

  • 購入者は作品を所有できる
  • 作成者は動画に関して制約が少ない（Web配信サイトではGIF動画などに強い制約がある）
  • 手数料が低いプラットフォームで販売できる
  • さまざまな販売サイトに気軽にもっていける
  • Web配信サイトより独自の本のデザインにできる
  • Markdownで執筆し、GitHubで管理可能

など、多くのメリットがあります。Googleスプレッドシート自体が基本PCで使わないと使えない部分があるため、今回はHtml電子書籍だけですが、もちろん、モバイル電子書籍も作っています。

出版業界では初心者向けの本が多いため、中級者向けの同人誌やマニアックな専門書がもう少し増えると、個人的には楽しみが増えると感じています。動画での情報収集は時間がかかるので、活字メインでショート動画が併用されているような書籍があればいいなと思っています。

現在、簡単なコマンド一発で、手軽かつ素早くマニュアルや電子書籍を生成できるようになっています。もし興味がある方が多ければ、技術同人誌としてテンプレートも含めてシェアしてもいいかなという気がしています。
